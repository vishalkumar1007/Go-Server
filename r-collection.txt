# GitHub API to Fetch Current Workflow Run Status

# purpose : This API fetches the detailed status of a specific workflow run in GitHub Actions.

api-endpoint : https://api.github.com/repos/<owner>/<repo>/actions/runs/<run_id>
request : GET
Authorization : choose Bearer Token and add your GitHub token.
